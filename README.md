An AI-powered GitHub App that automatically reviews pull requests, highlights issues, and suggests improvements.

## Features

* **Inline Review Comments** on bugs, edge cases, and improvements.
* **Test Suggestions** based on changed code.
* **Clear Explanations** that support developers of all levels.
* **Consistent Review Standards** across the team.

## Why Use It

* Speeds up code review cycles.
* Reduces reviewer workload.
* Improves code quality and consistency.
* Helps teams move quickly without sacrificing rigor.

## How It Works

1. A pull request is opened or updated.
2. The bot analyzes the diff and relevant context.
3. It posts comments directly in the PR.
